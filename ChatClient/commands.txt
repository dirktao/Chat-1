Build:
-std=c++0x -lpthread -lncurses


Start:
[ChatClient] [server] [port]


Commands:

/ban <nick>
Ban user

/deop <nick>
Remove operator rights from user

/disconnect [<message>]
Disconnects from server

/names
List of the users

/msg <nick> <message>
Private message

/nick <nick>
Change nick

/op <nick>
Give operator rights to user

/ops
List of operators

/unban <ip>
Remove bans

/who <nick>
Users host
